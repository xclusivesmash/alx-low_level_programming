# C - Pointers, Arrays, and Strings

### Description
Understanding C tools that can be used for memory management and data structures. This repository was created for learning C concepts at ALX SE programme.

### Requirements
* Languages: C
* OS: Ubuntu 20.04 LTS
* Style Guide: <a href="https://github.com/holbertonschool/Betty/wiki">Betty</a>

### Concepts
**pointers, arrays, and data structures**
* What are pointers and how to use them.
* What are arrays and how to use them.
* Differences between pointers and arrays.
* How to use strings.
* how to manipulate strings.
* Scope of variables.

### Script functionality
1. **0-reset_to_98.c** - takes a pointer to ```int``` as parameter and updates the value it points to to 98.<br>
2. **1-swap.c** - swops the values of two integers.<br>
3. **2-strlen.c** - finds the length of a string.<br>
4. **3-puts.c** -prints a string, followed by newline, to ```stdout```.<br>
5. **4-print_rev.c** - prints a string in reverse followed by a newline.<br>
6. **5-rev_string.c** - reverses a string.<br>
7. **6-puts2.c** - prints every other character of a string, starting with the first character, followed by a newline.<br>
8. **7-puts_half.c** - prints half of a string, followed by newline.<br>
9. **8-print_array.c** - prints *n* elements of an array of integers, followed by a newline.<br>
10. **9-strcpy.c** - copies the string pointed to by *src*, including the terminating null character, to buffer poited to by *dest*.<br>
