# C - Structures, typedef

### Project Description
As part of learning programming through the C programming language, structures and typedefs are important C tools that can be used in a variaty of applications to create user-defined types (with mixed data types) or to give synonyms to most data types.

### Technologies
* Languages: C
* OS: Ubuntu 20.04 LTS
* Style guide: <a href="https://github.com/holbertonschool/Betty/wiki">Betty Style</a>
* Allowed editors: vi, vim, or emacs.

### Concepts
* What are structures, when, why, and how to use them.
* How to use ```typedef```.

### Scripts
1. **dog.h** - defines a new type ```struct dog``` with the element with some elements.<br>
2. **1-init_dog.c** - initializes a variable of type ```struct dog```.<br>
3. **2-print_dog.c** - prints a ```struct dog```.<br>
4. **dog.h** - defines a new type ```dog_t``` as a new name for the type ```strcut dog```.<br>
5. **4-new_dog.c** - creates a new dog type.<br>
6. **5-free_dog.c** - frees dogs.<br>
