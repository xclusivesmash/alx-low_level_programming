# C - File I/O

### Description
File handling is an important part of any programming language. C offers file handling through system call. As part of learning programming through the C programming language, a directory was created with scripts to solved drills at ALX SE Programme.

### Technologies
* Language(s): C
* OS: Ubuntu 20.04 LTS
* Style guide: <a href="https://github.com/holbertonschool/Betty/wiki">Betty Style</a>
* Allowed editors: ```vi```, ```vim```, or ```emacs```

### Concepts
* How to create, open, close, read and write files.
* What are file descriptors.
* What are the standard file descriptors, their purpose, and their ```POSIX``` names.
* How to use the I/O system calls ```open```, ```close```, ```read```, and ```write```.
* What are and how to use the flags ```O_RDONLY```, ```O_WRONLY```, and ```O_RDWR```.
* What are file permissions.
* How to set file permissions when creating a file with the ```open``` system call.
* What is a system call.
* What is the difference between a function and a system call.

### Scripts
1. **0-read_textfile.c** - reads a text file and prints it to the ```POSIX``` standard output.<br>
2. **1-create_file.c** - creates a file.<br>
3. **2-append_text_to_file.c** - appends text at the end of a file.<br>
4. **3-cp.c** - copies the contents of a file to another file.<br>
5. **100-elf_header.c** - displays the information contained in the ```ELF``` header at the start of the ```ELF``` file.<br>
