# C - Singly Linked Lists

### Description
Linked Lists are an important integral of data structures that are more usefull to most applications. This directory contains scripts with solutions to tasks as a basis for learning more about C programming language at ALX.

### Technologies
* Language(s): C
* OS: Ubuntu 20.04 LTS
* Style guide: <a href="https://github.com/holbertonschool/Betty/wiki">Betty Style</a>
* Allowed editors: vi, vim, or emacs

### Concepts
* What are linked lists.
* When and why should one use linked lists over arrays.
* How to build and use linked lists.

### Scripts
1. **0-print_list.c** - prints elements of a linked list.<br>
2. **1-list_len.c** - returns the numbers of elements of a linked list.<br>
3. **2-add_node.c** - adds a new node at the beginning of a linked list.<br>
4. **3-add_node_end.c** - adds a new node at the end of a linked list.<br>
5. **4-free_list.c** - frees a linked list in memory.<br>
6. **100-first.c** - prints some text beofore the main function is executed.<br>
7. **101-hello_holberton.asm** - writes a 64-bit program in assembly that prints some text, followed by a newline.<br>
