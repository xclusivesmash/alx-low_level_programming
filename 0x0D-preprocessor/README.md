# C - Preprocessor

### Project Description
The project is about C preprocessor directives and how to use them in code achieve certain tasks. It emphasizes on the use of macros, what directives mean, and the disadvantages of using macros in code.

### Requirements
* Languages: C
* OS: Ubuntu 20.04 LTS
* Style guide: <a href="https://github.com/holbertonschool/Betty/wiki">Betty Style</a>
* Allowed editors: vim, vi, or emacs.

### Concepts
* What are macros and how to use them.
* What are the most common predefined macros.
* How to include guard your hearder files.

### Scripts
1. **0-object_like_macro.h** - creates a header file that defines a macro named ```SIZE``` as an abbreviation for the token ```1024```.<br>
2. **1-pi.h** - creates a header file that defines a macro named ```PI``` as an abbreviation for the token ```3.14159265359```.<br>
3. **2-main.c** - prints the name of the file the program was compiles from, followed by newline.<br>
4. **3-function_like_macro.h** - a function like macro ```ABS(x)``` that computes an absolute value of a number ```x```.<br>
5. **4-sum.h** - a funciton like macro ```SUM(x, y)``` that computes the sum of ```x``` and ```y```.<br>
