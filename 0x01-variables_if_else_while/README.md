# C - Variables, if, else, while

### Description
The projects focuses on the basic C flow control constructs. This repository was created for learning the C programming language at ALX SE programme.

### Requirements
* Languages: C
* OS: Ubuntu 20.04 LTS
* Style Guide: <a href="https://github.com/holbertonschool/Betty/wiki">Betty</a>
* Editors: ```vi```, ```vim```, ```emacs```.

### Concepts
* What are arithmetic operators and how to use them.
* What are logical operators and how to use them.
* What are the relational operators and how to use them.
* What values are considered TRUE and FALSE in C.
* What are boolean operators and how to use them.
* How to use ```if```, ```if ... else``` statements.
* How to use comments.
* How to declare variables of types ```char```, ```int```, ```unsigned int```.
* How to assign values to variables.
* How to print values of variables of type ```char```, ```int````, ```unsigned int``` using ```printf```.
* How to use the ```while``` loop.
* How to use variables with the ```while``` loop.
* How to print variables using ```printf```.
* What is the ```ASCII``` character set.
* What is the purpose of the ```gcc``` flags ```-m32``` and ```-m64```.

### Script functionality
0. **0-positive_or_negative.c** - Assigns a random number to variable n each time the code is executed, then prints whether n is positive, zero, or negative.<br>
1. **1-last_digit.c** - Assigns a random number to n and prints the last digit of n.<br>
2. **2-print_alphabet.c** - Prints alphabets in lowercase, followed by a new line.<br>
3. **3-print_alphabets.c** - Prints alphabets in uppercase, followed by newline.<br>
4. **4-print_alphabt.c** - Prints all letters in lowercase, except for letter 'q' and 'e', followed by newline.<br>
5. **5-print_numbers.c** - Prints all single digit number of the base 10 system, starting from 0, followed by newline.<br>
6. **6-print_numberz.c** - Prints all single digit numbers of the base 10 system, starting from 0, followed by newline, and only putchar can be used.<br>
7. **7-print_tebahpla.c** - Prints lowercase alphabets on reverse, followed by newline.<br>
8. **8-print_base16.c** - Prints all numbers of base 16 in lowercase, follwed by newline.<br>
9. **9-print_comb.c** - Prints all possible combinations of a single-digit number.<br>
10. **100-printcomb3.c** - Prints all possible combinations of two-digits. Numbers my be separated by , followed by space. The two digits must be different. The numbers must be printed in ascending order.<br>
11. **101-print_comb4.c** - Prints all possible different combinations of three digits. The numbers must be separated by , follwed by space. The three digits must be different. Only putchar can be used.
12. **102-print_comb5.c** - Prints all combinations of two two-digit numbers. The numebrs should range from 0 to 99. The two numbers should be separated by space.The combination of numbers mjust be separated by , followed by space. You can only use putchar for a maximum of 8 times in your code. You are not allowed to use any variable of type char in your code.<br>
