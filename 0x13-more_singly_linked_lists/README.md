# C - More singly linked lists

### Description
Linked lists are an important tool in data structure. As part of the ALX SE programme, this repository is created to learn more on linked lists manipulation through a set of predefined exercises.

### Technologies
* Language(s): C
* OS: Ubuntu 20.04 LTS
* Style guide: <a href="https://github.com/holbertonschool/Betty/wiki">Betty Style</a>
* Allowed editiors: ```vi, vim, or emacs```

### Concepts
* What are linked lists.
* How to use linked lists.
* How to look for the right source of information without too much help.

### Scripts
1. **0-print_listint.c** - prints all the elements of a linked list.<br>
2. **1-listint_len.c** - returns the number of elements in a linked list.<br>
3. **2-add_nodeint.c** - adds a new node at the beginning of a linked list.<br>
4. **3-add_nodeint_end.c** - adds a new node at the end of a linked list.<br>
5. **4-free_listint.c** - frees a linked list.<br>
6. **5-free_listint2.c** - frees a linked list.<br>
7. **6-pop_listint.c** - deletes the head node of a linked list, and returns the head node's data.<br>
8. **7-get_nodeint.c** - returns the nth node of a linked list.<br>
9. **8-sum_listint.c** - returns the sum of all the data of a linked list.<br>
10. **9-insert_nodeint.c** - inserts a new node at a given position.<br>
11. **10-delete_nodeint.c** - delets a node at a given index of a linked list.<br>
12. **100-reverse_listint.c** - reverses a linked list.<br>
13. **101-print_listint_safe.c** - prints a linked list.<br>
14. **102-free_listint_safe.c** - frees a linked list (safe version).<br>
15. **103-find_loop.c** - finds a loop in a linked list.<br>
