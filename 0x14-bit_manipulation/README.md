# C - Bit manipulation

### Description
Bit manipulation is important in computing because computers only understand the language of zeros and ones. As a result of learning programming through the C programming language, this repository was created.

### Technologies
* Language(s): C
* OS: Ubuntu 20.04 LTS
* Style guide: <a href="https://github.com/holbertonschool/Betty/wiki">Bettty Style</a>
* Allowed editors: vi, vim, or emacs

### Concepts
* Different types of bit manipulation operators
* How to manipulate bits

### Scripts
1. **0-binary_to_uint.c** - converts a binary number to an unsigned int.<br>
2. **1-print_binary.c** - prints abinary representation of a number.<br>
3. **2-get_bit.c** - returns a value of a bit at a given index.<br>
4. **3-set_bit.c** - sets the value of a bit to 1 at a given index.<br>
5. **4-clear_bit.c** - sets the value of a bit to 0 at a given index.<br>
6. **5-flip_bits.c** - returns the number of bits you would need to flip to get from one number to another.<br>
7. **100-get_endianness.c** - checks the endianness.<br>
8. **101-password** - finds a password for a certain program.<br>
