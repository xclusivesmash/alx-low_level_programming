# C - Static Libraries

### Requirements
* Languages: C, Bash<br>
* OS: Ubuntu 20.04 LTS<br>
* Style guidelines: <a href="https://github.com/holbertonschool/Betty/wiki">Betty Style</a><br>

### Description
The project is created for learning the C programming language at the Holberton School/ALX SE programme.<br>

### Concepts
* What is a Static Library?<br>
* How to create a static library?<br>
* How to use a static library?<br>
* What do the commands ```ar -rc```, ```ranlib```, and ```nm``` mean?<br>

### File functionality
1. **main.h** - header file containing function prototypes of functions in the library.<br>
2. **libmy.a** - static library containing object files.<br>
3. **create_static_lib.sh** - bash file used for creating a static library.<br>
