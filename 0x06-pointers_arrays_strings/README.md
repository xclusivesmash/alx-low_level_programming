# C - More pointers, arrays, and strings

### Description
This repository contains solutions to problems based on pointers, arrays, and strings. This is part of learning C concepts through the ALX SE Programme.

### Requirements
* Languages: C
* OS: Ubuntu 20.04 LTS
* Style Guide: <a href="https://github.com/holbertonschool/Betty/wiki">Betty</a>

### Concepts
* What are pointers and how to use them.
* What are arrays and how to use them.
* The differences between pointers and arrays.
* How to use strings.
* How to manipulate strings.
* Scope of variables.

### Script functionality
0. **0-strcat.c** - Create a function that contatenates two strings. The script contains a function that appends the *src* string to the *dest* string, overwriting the terminating null byte at the end of *dest*, and then appends the terminating null byte.<br>
1. **1-strncat.c** - concatenates two strings by using at most *n* bytes from *src*.<br>
2. **2-strncpy.c** - copies a string.<br>
3. **3-strcmp.c** - compares two strings.<br>
4. **4-rev_array.c** - reverses the contents of an array of integers.<br>
5. **5-string_toupper.c** - changes all lowercase letters of a string to uppercase.<br>
6. **6-cap_string.c** - capitalizes all words of a string.<br>
7. **7-leet.c** - encodes a string into 1337.<br>
8. **101-print_number.c** - prints an integer.<br>
