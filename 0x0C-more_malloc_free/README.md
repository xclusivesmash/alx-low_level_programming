# C - more malloc and free

### Description
This project is based on understanding memory allocation in C programming language and how things happen behind the scenes. 

### Requirements
* Languages: C
* OS: Ubuntu 20.04 LTS
* Editors: ```vi```, ```vim```, or ```emacs```
* Style guide: <a href="https://github.com/holbertonschool/Betty/wiki">Betty Style</a>

### Concepts
* Memory allocation using ```malloc``` in C
* Memory deallocation using ```free``` in C
* How to use the ```exit``` function in C
* What are the functions ```calloc``` and ```realloc``` from the standard library and how to use these functions.

### Scripts
1. **0-malloc_checked.c** - Allocates memory using ```malloc``` and reu=turns a pointer to the allocated memory.<br>
2. **1-string_concat.c** - contenates two strings. If the function fails, it should return ```NULL```. If ```n``` is greater or equal to the length of ```s2```, then use the entire string ```s2```. If ```NULL``` is passed, treat it as an empty string.<br>
3. **2-calloc.c** - allocates memory for an array using ```malloc```.<br>
4. **3-array_range.c** - creates an array of integers.<br>
5. **100-realloc.c** - reallocates a memory block using ```malloc``` and ```free```.<br>
6. **101-mul.c** - a program that multiplies two positive numbers.<br>
