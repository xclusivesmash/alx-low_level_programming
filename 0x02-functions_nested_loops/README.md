## C -Functions and nested loops

The project illustrates the idea of function declaration, function definition, return types of functions, parameters passed to functions. It also emphasizes the use of various control structures (loops to be precise) nested together to achieve basic tasks.

0. Print *_putchar*, followed by a newline.<br>
1. Prints the alphabets in lowercase, followed by a newline.<br>
2. Prints 10 times the alphabets in lowercase, followed by a newline.<br>
3. Checks for lowercase charatcer. Returns 1 if *c* is lowercase, otherwise returns 0.<br>
4. Checks for alphabetic character. Return 1 if *c* is a letter, lowercase or uppercase, otherwise 0 is returned.<br>
5. Prints the sign of a number. Returns 1 and prints *+* if n is greater than 0. Returns 0 and prints *0* if n is zero. Returns -1 and prints *-* if n is less than 0.<br>
6. Computes the absolute value of an integer.<br>
7. Prints the last digit fof a number.<br>
8. Prints eery minute of the day of Jack Bauer, starting from 00:00 to 23:59.<br>
10. Adds two integers and returns the result.<br>
11. Prints all natural numbers from n to 98, followed by a newline. The numbers myst be separated by a comma followed by space. The last printed number should be 98.<br>
