# C -Functions and nested loops

### Description
The project illustrates the idea of function declaration, function definition, return types of functions, parameters passed to functions. It also emphasizes the use of various control structures (loops to be precise) nested together to achieve basic tasks.

### Requirements
* Languages: C
* OS: Ubuntu 20.04 LTS
* Style Guide: <a href="https://github.com/holbertonschool/Betty/wiki">Betty</a>
* Editors: ```vi```, ```vim```, ```emacs```

### Concepts
* What are nested loops and how to use them.
* What is a function and how do you use a function.
* Difference between a declaration and definition of a function.
* What is a prototype.
* Scope of variables.
* What are the ```gcc``` flags ```-Wall```, ```-Werror```, ```-Wextra```, ```-pedantic```, ```std=gnu89```
* What are header files and how to use them with ```#include```.

### Script functionality
0. **0-putchar.c** - Print *_putchar*, followed by a newline.<br>
1. **1-alphabets.c** - Prints the alphabets in lowercase, followed by a newline.<br>
2. **2-print_alphabet_x10.c** - Prints 10 times the alphabets in lowercase, followed by a newline.<br>
3. **3-islower.c** - Checks for lowercase charatcer. Returns 1 if *c* is lowercase, otherwise returns 0.<br>
4. **4-isalpha.c** - Checks for alphabetic character. Return 1 if *c* is a letter, lowercase or uppercase, otherwise 0 is returned.<br>
5. **5-sign.c** - Prints the sign of a number. Returns 1 and prints *+* if n is greater than 0. Returns 0 and prints *0* if n is zero. Returns -1 and prints *-* if n is less than 0.<br>
6. **6-abs.c** - Computes the absolute value of an integer.<br>
7. **7-print_last_digit.c** - Prints the last digit fof a number.<br>
8. **8-24_hours.c** - Prints eery minute of the day of Jack Bauer, starting from 00:00 to 23:59.<br>
10. **10-add.c** - Adds two integers and returns the result.<br>
11. **11-print_to_98.c** - Prints all natural numbers from n to 98, followed by a newline. The numbers myst be separated by a comma followed by space. The last printed number should be 98.<br>
