# C - Hello, World

### Description
The project focuses on basic C concepts namely, compilation process, creating a C program following the betty-style and betty-doc.

### Requirements
* Languages: C, Bash
* OS: Ubuntu 20.04 LTS
* Style Guide: <a href="https://github.com/holbertonschool/Betty/wiki">BettyStyle</a>
* Editors: ```vi```, ```vim```, or ```emacs```

### Concepts
* C programmming language
* Compilation process in C
* The Betty Style
* Entry points
* What is the ```main``` function.
* Who created C
* Linus Torvalds

### Script functionality
1. **0-preprocessor** - runs a C file through the preprocessor and saves the results in another file.<br>
2. **1-compiler** - Compile the C file but does not link. The C file will be saved in the variable name ```$CFILE```.<br>
3. **2-assembler** - generates the assembly code of a C code and save the results in an output file.<br>
4. **3-name** - compile a Cfile and create an executable named ```cisfun```.<br>
5. **4-puts.c** - prints exactly ```"Programming is like building a multilingual puzzle.```, followed by newline.<br>
6. **5-printf.c** - prints exactly ```with proper grammar, but the outcome is a piece of art```, followed by newline.<br>
7. **6-size.c** - prints the sizes of various types on a computer it is compiled on.<br>
8. **100-intel** - generates the assembly code of a C code and save it in the output file.<br>
9. **101-quote.c** - prints exactly ```and that piece of art is useful" - Dora Korpar, 2015-10-19```, followed by newline, to standard error.<br>
