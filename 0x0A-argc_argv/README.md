# C - argc, argv

### Descritpion
Files contained within this repository are used for learning more about C argc and argv arguments to main as a requirement for learning low-level programming concepts using C.

### Requirements
* Languages: C
* OS: Ubuntu 20.04 LTS
* Style Guide: <a href="https://github.com/holbertonschool/Betty/wiki">Betty Style</a>
* Editors: ```vi```, ```vim```, or ```emacs```

### Concepts
* Arguments to main
* ```argc``` and ```argv```
* What does ```argc``` and ```argv``` mean.
* how to compile functions with unused variables or parameters.

### Script description
1. **0-whatsmyname.c** - prints the program's name, followed by newline. If you rename the program, it will print the new name without having to compile again.<br>
2. **1-args.c** - prints the number of arguments passed into it.<br>
3. **2-args.c** - prints all arguments it receives. Only print one argument per line, ending with a newline.<br>
4. **3-mul.c** - multiplies two numbers, print the result of the multiplication (followed by newline), and if the program does not receive two arguments, you should print ```Error```, followed by newline, and return ```1```.<br>
5. **4-add.c** - adds positive numbers and print results, followed by newline. If no number is passed to the program, print 0 followed by newline. If one of the numbers contains symbols that are not digits, print ```Error```, followed by newline, and return ```1```.<br>
6. **100-change.c** - prints the minimum number of coins to make change for an amount of money. If the number of arguments passed to the program is not exactly ```1```, print ```Error```, followed by newline, and return ```1```.<br>
